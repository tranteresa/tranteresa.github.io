Additional Improvements
Improving the Journey

User feedback and testing was essential to improving the product's overall experience. I implemented changes in two main areas to improve the user journey:

  • Navigation
  • Visual Cues


01 Map & List Views

The previous design forced users to choose between a list or map view. Only one view could be displayed at a time and users had to toggle between views by pressing an icon.

I improved the accessibility by simultaneously displaying both views on one screen. This overlapping design reduces the number of clicks required for users to gather information. Swiping up will bring users to a full list view and swiping down will bring users to a full map view.

Dual Map & List View.

02 Interactive Map

If users want to access information through a visual map or do not have time to scroll through the list view, they can use the interactive map to zoom in or out to find businesses in their area. Users can also click on location pins to preview the item.

Interactive Map.

03 Main Tab Navigation

The tab navigation offers three types of content: personalized location-based, bookmarked items, and previously searched.

Tab Style Navigation.

04 Category Icons

Color coded category icons were created and added to each list item. This helped improve information processing. Users can quickly scan list items and select the item that meets their needs.

Color coded category icons.
